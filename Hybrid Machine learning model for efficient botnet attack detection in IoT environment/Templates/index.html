ABOUT

This platform leverages **Transformer-Based Classification** to significantly enhance the detection of **social bots**. By fine-tuning large language models on social media data, we achieve a deeper understanding of linguistic and behavioral patterns, leading to highly accurate and robust bot identification.

Modules

Admin

The admin has secure access to the platform with a dedicated login. Once logged in, the admin can upload datasets, and oversee the system to ensure smooth operation. This role ensures that the platform has accurate information for bot detection model training.

Model Builder

The model builder logs in for fine-tuning the Transformer model. They preprocess social media datasets, manage the training and validation of the classification model, and deploy the highly-accurate bot detection system. This role is crucial for maintaining the model's effectiveness against evolving bot strategies.

User

Users can register and log in to the platform to utilize the bot detection service. Once logged in, they can input user profile or post data for instant analysis. The system uses the transformer model to classify the input, empowering users to identify and mitigate social bot influence.
